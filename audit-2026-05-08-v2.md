# Audit Leerkwartier — 12 perspectieven (2026-05-08, ronde 2)

12 review-agents + 1 screenshots-agent op codebase + live `https://leerkwartier.app/`. Screenshots in `C:\Users\mark-\AppData\Local\Temp\audit-2026-05-08-v2\screens\`.

## Executive summary

Leerkwartier is **technisch solide** maar lijdt aan een **identiteitsconflict**: de marketing zegt "Cito-koning", de UI is een speeltuin, en de Cito-content dekt ~40% van een echte eindtoets. Vier herhaalde signalen uit de meeste agents:

1. **Homepage faalt voor de gekozen ICP** (bezorgde Cito-ouder)
2. **Cito-content heeft één groot inhoudelijk gat: begrijpend-lezen-oefenteksten = 0%**
3. **Mobile-UX en eerste-laad-performance lossen elkaars problemen NIET op** (2.5MB eager bundle + 100vh-bug)
4. **RLS-policies zijn belangrijker dan welke security-fix dan ook**

## Cross-cutting bevindingen

### 1. Cito-content is voor 60% "leuk-leren-met-label"
- Begrijpend lezen: 0 echte oefen-teksten van 250+ woorden (alleen meta-strategie)
- Taalverzorging: geen leestekens-pad, geen "welke zin is correct"-vorm
- Vraagstijl ~3-5× te kort vs echte Cito (kale sommen i.p.v. redactie)
- Dummy-antwoorden i.p.v. plausibele afleiders ("40 jaar" als optie bij "wat is 40%?")
- **Verdict**: rekendeel = solide 1F, taaldeel = gebrekkig, studievaardigheden = oppervlakkig

### 2. Homepage botst met ICP-keuze
- "v3 design"-eyebrow, OBLITERATOR-game, AI-vragen-tegel, ticker met namen → speeltuin-signaal
- Geen testimonials, geen "X kinderen oefenen", geen "wie is Mark"-pagina
- Prijs €1,95 voelt verdacht goedkoop (Squla €5, IEP-boek €15) en staat op "BINNENKORT"
- Ouder-dashboard is dark-mode game-skin, geen rapportkaart
- Twee modals stapelen op homepage (age-gate + video-promo)

### 3. Privacy + Security: code is OK, RLS is achilleshiel
- `KindAcceptBanner` RLS: kind kan andermans `verified` flippen via naam-rade
- AI-endpoints (`/api/tutor-chat`, `/api/generate-questions`, `/api/preview-topic`): geen rate-limit, geen origin-check → API-key drain mogelijk
- Feedback-screenshots bucket nog public-URL i.p.v. signed (privacy.html-belofte gebroken)
- `dangerouslySetInnerHTML` voor SVG met AI-output zonder DOMPurify
- `deleteAllMyData` mist `profiles, quizzes, upgrade_waitlist, learn_path_waitlist`
- AgeGate puur symbolisch (localStorage clear omzeilt)

### 4. Performance + Mobile = combo-killer
- `data-learnpaths` 2.5MB (662KB gzip) wordt EAGER geladen — `App.jsx:69` static import
- Cover-images 20MB ongecompresseerd, 0 webp
- `100vh` op 18 plekken → iOS Safari bottom-bar gobble (60-100px witruimte)
- `viewport: maximum-scale=1.0, user-scalable=no` in `index.html` → blokkeert pinch + a11y-flag
- Touch-targets <44px op chips, pillen 15/30/50, v1/v2-toggle
- Scroll-trap in `CitoLeerpadToets` (fixed bottom-bar zonder safe-area-inset)

### 5. A11y is systemisch, niet per-component
- Inline-style architectuur = `:focus-visible` niet bereikbaar zonder global stylesheet
- 7 inputs hebben `outline: none` zonder vervanger
- 25+ `<label>` zonder `htmlFor` gekoppeld
- `CitoLeerpadToets` totaal kaal: geen `role="timer"`, geen `aria-live`, geen `<main>`
- OBLITERATOR: 0 `prefers-reduced-motion`-checks (tokens.css raakt canvas niet)

### 6. Code-architectuur: prop-drilling op 1 niveau werkt zolang je solo bent
- App.jsx 1611 regels, 25+ useStates, 0 Context-Providers
- Option-shuffler 3× gedupliceerd (`App.jsx:558`, `LearnPath.jsx:254`, `CitoLeerpadToets.jsx:65`)
- 169 tests, allemaal pure-utility — 0 tests op `LearnPath`/`CitoLeerpadToets`/`KindAcceptBanner`/`PlayQuiz`/`ObliteratorGame`/`App`
- Quiz-id-prefix-routing (`startsWith("self-tafels")` etc.) op 8 plekken — 1 helper kan dat fixen

### 7. Concrete bugs die in screenshots opvielen
- **Markdown-asterisks worden niet gerendered** in `CitoLeerpadToets` toetsvragen ("Verleden tijd van \*\*fietsen\*\*" toont sterren letterlijk)
- Homepage-knop **"📈 Mijn voortgang — Audit"** bevat dev-jargon "Audit" (zichtbaar voor gebruikers)
- Default 30 vragen op mini-eindtoets — botst met 15-min-belofte
- 2× `406 Failed to load` op `/subscriptions?...` op elke route (RLS of `single()` op lege rij)

## Top 5 grotere refactors (>2u)

1. **`ALL_LEARN_PATHS` lazy maken** — App.jsx:69 → `useEffect`-dynamic-import in `MeeBezig.jsx`. Bespaart 662KB gzip op first-load. **Grootste hefboom in hele codebase.**
2. **Begrijpend-lezen-paden bouwen** — 8 leerpaden met echte teksten van 250+ woorden + 4 vragen elk. Sluit het grootste Cito-gat (25% van eindtoets, nu 0% gedekt).
3. **RLS-policy review-doc** — per tabel (`leaderboard`, `parent_child_links`, `topic_mastery`, `hall_of_fame`, `obliterator_scores`) policy + comment + test. Eenmalig 2u, lost #1 security en #1 privacy gat in één.
4. **Cover-images batch naar .webp** — 20MB → ~2MB sitewide. ~200-800KB per pagina.
5. **Context-laag voor user/quiz/persistence** — 3 hooks (`useUser`, `useQuizFlow`, `useAppPersistence`) maken Sprint D+ losse routes mogelijk zonder App.jsx aan te raken.

## Strategische keuzes voor Mark

Drie beslissingen die alleen jij kunt nemen:

1. **OBLITERATOR-game**: Cito-ouder ICP wil GEEN Geometry-Dash bij €1,95/mnd. Verstoppen achter `/spelen` of zichtbaar laten? (Nu duidelijk afleider voor ICP, maar mogelijk Pim-retention-trekker.)
2. **VO-leerpaden** (klas 3-4 + bovenbouw, ~40 paden): laten staan op disk maar uit homepage/marketing halen, of helemaal ophouden? Buiten ICP, kost wel onderhoud.
3. **Prijs**: €1,95 voelt te goedkoop ("alpha"). Squla €5, IEP €15. Voorstel: €4,95/mnd of €39/jaar; haal "BINNENKORT"-badge weg, zet iDEAL aan.
