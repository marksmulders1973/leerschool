# 🧠 LeerSchool

Samen slim worden — op school en thuis!

## Deployen op Vercel (gratis)

### Stap 1: GitHub account
- Ga naar [github.com](https://github.com) en maak een gratis account aan (als je dat nog niet hebt)

### Stap 2: Upload naar GitHub
- Klik op **"New repository"** (het + icoontje rechtsboven)
- Noem het `leerschool`
- Klik **"Create repository"**
- Upload alle bestanden uit deze map naar de repository

### Stap 3: Vercel
- Ga naar [vercel.com](https://vercel.com) en log in met je GitHub account
- Klik **"Add New → Project"**
- Selecteer je `leerschool` repository
- Vercel herkent automatisch dat het een Vite project is
- Klik **"Deploy"**
- Klaar! Je krijgt een URL zoals `leerschool.vercel.app`

### Stap 4: Delen
- Deel je URL via WhatsApp met de knoppen in de app
- Iedereen kan de app openen op telefoon, tablet of computer!

## Lokaal testen
```bash
npm install
npm run dev
```
Open `http://localhost:5173` in je browser.
