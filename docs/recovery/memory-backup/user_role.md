Mark Smulders (Mark-smulders@hotmail.com) is solo-ontwikkelaar van **Studiebol**, een leer-/quiz-app voor Nederlandse leerlingen (basisschool + voortgezet onderwijs).

**Hoe hij werkt:**
- Werkt in Nederlands. Antwoord in het Nederlands, tenzij hij Engels gebruikt.
- Geen dev-team — hij is zowel product, design, als developer. Houd uitleg praktisch en hands-on.
- Werkt vaak vanaf zijn thuismap `C:\Users\mark-` met Claude Code vanaf een desktop-shortcut.
- Doel: commerciële lancering via Play Store, minimaal €200/maand inkomen als eerste mijlpaal.

**Implicaties voor samenwerking:**
- Leg technische beslissingen uit in termen van *wat het kost* en *wat het oplevert* (tijd/geld).
- Default naar gratis of goedkope oplossingen (hij koos bewust Vercel Hobby + TransIP domein + Gemini fallback).
- Wees proactief met herstel van context bij session-breaks — hij raakt snel het overzicht kwijt tussen sessies.
